## Request flow

```
HTTP Request
    │
    ▼
requestLogger      (middleware/requestLogger.js)
    │
    ▼
Route match        (resources/<name>/<name>.routes.js)
    │
    ▼
validate()         (middleware/validate.js) — only on writes
    │  populates req.validated
    ▼
Controller         (resources/<name>/<name>.controller.js)
    │  reads req.validated, req.params, req.query
    ▼
Service            (resources/<name>/<name>.service.js)
    │  throws AppError subclasses
    ▼
Store              (resources/<name>/<name>.store.js / store/createStore.js)
    │
    ▼
Response (JSON)
    │
    ▼ (on error)
errorHandler       (middleware/errorHandler.js)
```

## Data model

### Task

| Field | Type | Notes |
|---|---|---|
| `id` | UUID string | Auto-generated by store |
| `title` | string | Required. Trimmed. |
| `description` | string | Optional. |
| `status` | `todo` \| `in_progress` \| `done` | Default: `todo` |
| `priority` | `low` \| `medium` \| `high` | Default: `medium` |
| `tags` | string[] | Default: `[]`. Stored lowercase. |
| `dueDate` | ISO 8601 string \| null | Optional. |
| `assignee` | string \| null | Optional. |
| `createdAt` | ISO 8601 string | Auto-set by store |
| `updatedAt` | ISO 8601 string | Auto-updated by store on every edit |

## Pagination

All list endpoints support:

| Query param | Default | Max | Notes |
|---|---|---|---|
| `page` | 1 | — | 1-indexed |
| `limit` | 20 | 100 | Clamped at 100 |

Response shape for lists:
```json
{
  "data": [...],
  "meta": {
    "page": 1,
    "limit": 20,
    "total": 42,
    "pages": 3
  }
}
```

## Error response shape

```json
{ "error": "Human-readable message" }
{ "error": "Validation failed", "errors": ["field is required", "..."] }
```

## Store API

`createStore()` returns an object with five methods:

| Method | Signature | Notes |
|---|---|---|
| `list()` | `() → record[]` | All records as an array |
| `get(id)` | `(string) → record \| null` | Null if missing |
| `create(data)` | `(object) → record` | Adds id, createdAt, updatedAt |
| `update(id, data)` | `(string, object) → record \| null` | Merges, bumps updatedAt |
| `remove(id)` | `(string) → boolean` | True if deleted |
